README

Folder Contents:
- students_db.sql: used to populate database.
- COMP3005_asmt3\src\main\java\Main: Source code

Steps to run and compile Application:
- Open pgAdmin, create a new Database and then populate it with the students_db.sql file.
- In Main class, update the url, user, and password class variables accordingly.
- In main(), uncomment and modify the parameters of whichever function you would like to test.
- Run Main.main().
